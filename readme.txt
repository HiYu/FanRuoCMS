1、本系统的0.1版本未实现安装程序故需用户手动导入数据库
2、若数据库名称不是本系统默认的fanruo的话，需修改Home和admin两个项目的配置文件，配置文件在相应项目目录下的Conf文件夹下的conf.php
   文件中，将'DB_DSN' => 'mysql://root:@localhost:3306/fanruo',修改为自己的数据库信息
3、按照以上两步即可安装完成FanRuoCMS 0.1 但进入后台进行网站管理需在前天注册页面注册管理员，注意管理员只有一个，企图注册
   两个或以上管理员将失败。
4、通过前台登录窗口将提示失败，为了网站安全隐藏了后台登陆窗口，用户可在地址栏中输入：yourdomain/admin.php进入后台登陆页面
